# Personal Finance Behaviour — Statistical Validation

A business-intelligence analysis of personal financial behaviour, using statistical
hypothesis testing to validate patterns in spending, saving, and income data. Built
as a Jupyter notebook with a Django-based component for serving the analysis.

## Highlights
- Exploratory data analysis of personal finance behaviour
- Statistical validation (hypothesis testing) of behavioural patterns
- Reproducible, well-documented Jupyter workflow

## Tech
Python · pandas · Jupyter · statistical testing · Django

## Files
- `BI_Personal_Finance_Behaviour_Statistical_Validation_Django.ipynb` — full analysis notebook
